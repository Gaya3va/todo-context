# Todo Application Using Context API

## Objective
To build a Todo application using React Context API where all todo data and actions are managed globally.

---

## Features
- Add Todo
- Delete Todo
- Global state using Context API
- No props drilling

---

## Context Details
TodoContext manages:
- todos
- addTodo
- deleteTodo

---

## Rules Followed
- No todo state inside components
- Context API manages all state
- Clean component separation

---

## Folder Structure
src/
- context/
- components/
- App.js
- index.js

---

## How to Run
npm install  
npm start
